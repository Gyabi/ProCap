# Requirement
- tauri 1.4.0
- nextjs 14.0.3
- typescript ^5
- tailwindcss 3.3.0

# Installation
リリースよりインストーラのダウンロード<br>
またはローカルにクローンして以下コマンドを実行
```
npm install
cargo tauri build
```
